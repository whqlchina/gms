# gms
谷歌GMS认证（Google Mobile Services Test Certification ）是谷歌公司为了确保全世界的硬件厂商设计和生产的安卓为基础的各种智能硬件产品的兼容性、稳定性，以及这些硬件与安卓各种自带的各种软件APP产品和服务的兼容性、稳定性方面的测试和认证，包括安卓智能手机、安卓智能平板、安卓手表、安卓智能汽车、谷歌眼镜、人工智能、VR等。
